## 📌 What is StartSmart AI?

StartSmart AI is a national-level hackathon project that helps entrepreneurs, students, and startup founders **validate their startup ideas instantly** using AI-powered analysis.

Just type your idea — get a full business breakdown in seconds.

---

## ✨ Features

- ⚡ **AI-Powered Analysis** — Problem statement, target audience, SWOT, revenue model, marketing strategy, competitor analysis, risk factors, and growth suggestions
- 📊 **Interactive Charts** — Market growth bar chart, 3-year revenue projection, SWOT radar chart
- 🔐 **Google Authentication** — Firebase-powered secure login
- 📁 **User History** — Firebase Firestore saves all past analyses per user
- 📄 **PDF Export** — Download full analysis as a professional PDF report
- 💬 **Share** — Share results directly on WhatsApp and LinkedIn
- 📱 **Mobile Responsive** — Works on all devices

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/hari768/StartSmart-AI.git
cd StartSmart-AI
```

### 2. Add your API keys
In `script.js`, add your OpenRouter API key:
```js
const API_KEY = "your-openrouter-key-here";
```

### 3. Open with Live Server
Use VS Code Live Server extension or open `index.html` directly.
